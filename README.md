# 1.3.4-React
1)Ввод валидного email:


-Ввести в поле "user@domain.ru"

-Кликнуть по кнопке «Проверить» 

-Результат: Появляется всплывающее сообщение «Прекрасный email-адрес»


2) Отсутствует символ «@»:


-Ввести в поле "testmail.ru"
-Кликнуть по кнопке «Проверить»
-Результат: Под кнопкой отображается красное сообщение «Введен некорректный email»


3)Киррилица в локальной части:


-Ввести в поле "тест@mail.ru"
-Кликнуть по кнопке «Проверить»
-Результат: Под кнопкой отображается красное сообщение «Введен некорректный email»


4)Пустое поле:


-Оставить поле незаполненным
-Кликнуть по кнопке «Проверить»
-Результат: Под кнопкой отображается красное сообщение «Введен некорректный email»


5)Отсутствует доменная часть:


-Ввести в поле "test@"
-Кликнуть по кнопке «Проверить»
-Результат: Под кнопкой отображается красное сообщение «Введен некорректный email»


6)Несколько символов «@»:


-Ввести в поле «test@test@.ru»
-Нажать кнопку «Проверить»
-Результат: Под кнопкой появилось красное сообщение об ошибке «Введен некорректный email»


7)Две точки подряд в домене


-Ввести в поле "test@mail..ru"
-Кликнуть по кнопке «Проверить»
-Результат: Под кнопкой отображается красное сообщение «Введен некорректный email»


8) Email указан с знаком (! # $ % & ‘ * + — / =? ^ _ ` { | } ~) в username


-Ввести в поле email "test.kata@mail..ru"
-Нажать кнопку «Проверить»
-Результат:Появляется alert с текстом «Прекрасный email-адрес»


9)Email указан не латиницей в hostname


-Ввести в поле email "test@майл.ру"
-Нажать кнопку «Проверить»
-Результат: Выводится ошибка красного цвета над кнопкой «Проверить» с текстом «Введен некорректный email»
10)Отсутствует точка в домене
-Ввести в поле "test@mailru"
-Кликнуть по кнопке «Проверить»
-Результат: Под кнопкой отображается красное сообщение «Введен некорректный email»
